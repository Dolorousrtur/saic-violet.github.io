Data splits
 
 
Due to legal reasons we can not publish raw images from the dataset. However, we provide list of youtube videos we used along with frame indexes used for training and testing. We also publish parameters for SMPLx body model fitted using smplify-x mathod.
 
You may download a .zip file with the data splits and SMPLx parameters using the following links:
 

Data splits

SMPLx Parameters
 
 
The data splits .zip file contains following data:
  • TEDXPeople.json -- full list of video links and frame indexes splitted into train and test parts
  • TEDXPeople_upperbody.json -- subset containing only upperbody frames
  • TEDXPeople_512.json -- subset containing only frames where human body resolution is close to or exceeds 512p
 
.json files scheme
 
 
 
 
 
 
To get the data you need to download youtube videos with the identifiers stored in .json files and extract corresponding frames from them.
